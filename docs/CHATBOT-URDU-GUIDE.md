# Chatbot Patient Details Collection - Urdu Guide
# چیٹ بوٹ مریض کی تفصیلات اکٹھی کرنا - اردو گائیڈ

## خلاصہ (Overview)
یہ گائیڈ بتاتا ہے کہ چیٹ بوٹ (Riley) کیسے مریض کی معلومات اکٹھی کرتا ہے جب کوئی Clinica San Miguel میں اپائنٹمنٹ بک کرنا چاہتا ہے۔

## اہم اصول: ایک وقت میں ایک سوال (ONE QUESTION AT A TIME)

**اہم قاعدہ:** چیٹ بوٹ کو مریض کی تفصیلات ایک ایک کر کے پوچھنی چاہیے۔ کبھی بھی ایک سوال میں کئی معلومات نہ مانگیں۔

## قدم بہ قدم معلومات اکٹھی کرنا (Step-by-Step Collection)

### جب مریض کہے "میں اپائنٹمنٹ بک کرنا چاہتا ہوں"

چیٹ بوٹ یہ ترتیب فالو کرے گا:

### قدم 1: سلام کریں اور نام پوچھیں
```
چیٹ بوٹ: "ہیلو! میں Clinica San Miguel سے Riley ہوں۔ میں آپ کی اپائنٹمنٹ شیڈول کرنے میں مدد کروں گا۔ براہ کرم اپنا پورا نام بتائیں؟"

مریض: "جان سمتھ"
```

### قدم 2: فون نمبر پوچھیں
```
چیٹ بوٹ: "شکریہ جان۔ آپ کا بہترین فون نمبر کیا ہے؟"

مریض: "555-123-4567"
```

### قدم 3: تاریخ پیدائش پوچھیں
```
چیٹ بوٹ: "اور آپ کی تاریخ پیدائش کیا ہے؟ براہ کرم مہینہ، دن، سال کی شکل میں بتائیں۔"

مریض: "15 جنوری 1990"
```

### قدم 4: عمر پوچھیں
```
چیٹ بوٹ: "آپ کی عمر کتنی ہے؟"

مریض: "34"
```

### قدم 5: پتہ پوچھیں
```
چیٹ بوٹ: "آپ کا موجودہ پتہ کیا ہے؟"

مریض: "123 Main Street, Dallas, TX 75203"
```

### قدم 6: ای میل پوچھیں (اختیاری)
```
چیٹ بوٹ: "کیا آپ اپائنٹمنٹ کی یاد دہانی کے لیے ای میل ایڈریس دینا چاہیں گے؟"

مریض: "john.smith@email.com" یا "نہیں، ٹھیک ہے"
```

### قدم 7: اپائنٹمنٹ کی قسم پوچھیں
```
چیٹ بوٹ: "آپ کس قسم کی اپائنٹمنٹ چاہتے ہیں؟ ہم $19 میں مشاورت، $220 میں امیگریشن میڈیکل ایگزام، اور دیگر خدمات پیش کرتے ہیں۔"

مریض: "مجھے مشاورت چاہیے"
```

### قدم 8: تاریخ اور وقت پوچھیں
```
چیٹ بوٹ: "آپ کب آنا پسند کریں گے؟ کون سی تاریخ اور وقت آپ کے لیے بہتر ہے؟"

مریض: "کل دوپہر 2 بجے"
```

### قدم 9: تصدیق کریں اور اپائنٹمنٹ بنائیں
```
چیٹ بوٹ: "بہترین! آپ کی اپائنٹمنٹ کنفرم ہو گئی ہے۔ آپ کا کنفرمیشن کوڈ APT-000123 ہے۔ آپ کی مشاورت 10 دسمبر 2025 کو دوپہر 2:00 بجے شیڈول ہے۔ کیا میں کچھ اور مدد کر سکتا ہوں؟"
```

## چیٹ بوٹ کے لیے اہم قواعد

### ✅ کریں:
- سب سے پہلے مریض کا **نام** پوچھیں - یہ لازمی ہے
- ایک وقت میں ایک سوال پوچھیں
- اگلا سوال پوچھنے سے پہلے مریض کے جواب کا انتظار کریں
- بات چیت میں مریض کا نام استعمال کریں (مثلاً "شکریہ جان")
- اگر کوئی بات واضح نہ ہو تو تصدیق کریں
- دوستانہ، صبر سے، اور پیشہ ورانہ رویہ رکھیں
- تصدیق کے لیے معلومات دہرائیں

### ❌ نہ کریں:
- کبھی بھی ایک سوال میں کئی معلومات نہ مانگیں
- نام پوچھنا نہ چھوڑیں
- سوالات میں جلدی نہ کریں
- معلومات کا اندازہ نہ لگائیں
- جواب ملے بغیر آگے نہ بڑھیں

## غلط طریقہ کی مثال (کئی سوالات)
```
❌ غلط:
چیٹ بوٹ: "مجھے آپ کا نام، فون نمبر، تاریخ پیدائش، اور پتہ چاہیے۔"
```

## صحیح طریقہ کی مثال (ایک ایک کر کے)
```
✅ صحیح:
چیٹ بوٹ: "براہ کرم اپنا پورا نام بتائیں؟"
[جواب کا انتظار]
چیٹ بوٹ: "شکریہ۔ آپ کا بہترین فون نمبر کیا ہے؟"
[جواب کا انتظار]
چیٹ بوٹ: "اور آپ کی تاریخ پیدائش کیا ہے؟"
[اور اسی طرح...]
```

## ضروری مریض کی معلومات

### لازمی فیلڈز:
1. **پورا نام** (first_name, last_name)
2. **فون نمبر** (phone) - فارمیٹ: +1XXXXXXXXXX
3. **تاریخ پیدائش** (date_of_birth) - فارمیٹ: YYYY-MM-DD
4. **عمر** (DOB سے calculate یا براہ راست پوچھیں)
5. **پتہ** (ریکارڈ کے لیے)
6. **اپائنٹمنٹ کی تاریخ**
7. **اپائنٹمنٹ کا وقت**

### اختیاری فیلڈز:
1. **ای میل** (اپائنٹمنٹ کی یاد دہانی کے لیے)
2. **اپائنٹمنٹ کی قسم** (اگر نہ بتائی گئی تو "consultation" ڈیفالٹ)

## مختلف حالات کو سنبھالنا

### حالت 1: مریض ایک ساتھ کئی تفصیلات دے
```
مریض: "میرا نام جان سمتھ ہے اور میرا نمبر 555-1234 ہے"

چیٹ بوٹ: "شکریہ جان سمتھ۔ میرے پاس آپ کا فون نمبر 555-1234 ہے۔ اب، آپ کی تاریخ پیدائش کیا ہے؟"
```

### حالت 2: مریض واضح نہ ہو
```
مریض: "ام، میں تاریخ کے بارے میں یقین نہیں ہوں..."

چیٹ بوٹ: "کوئی بات نہیں۔ وقت لیں۔ آپ کی اپائنٹمنٹ کے لیے کون سی تاریخ بہتر ہوگی؟"
```

### حالت 3: مریض پہلے قیمت پوچھے
```
مریض: "مشاورت کی قیمت کتنی ہے؟"

چیٹ بوٹ: "مشاورت کی قیمت $19 ہے۔ ہم میڈیکل انشورنس قبول نہیں کرتے، لیکن ہماری قیمتیں بہت سستی ہیں۔ کیا آپ مشاورت شیڈول کرنا چاہیں گے؟"

مریض: "جی ہاں"

چیٹ بوٹ: "بہترین! براہ کرم اپنا پورا نام بتائیں؟"
```

## ڈیٹا بیس کے ساتھ انضمام

تمام معلومات اکٹھی کرنے کے بعد، چیٹ بوٹ:

1. **چیک کرتا ہے** کہ مریض ڈیٹا بیس میں موجود ہے (فون نمبر سے)
2. **نیا مریض ریکارڈ بناتا ہے** اگر وہ موجود نہیں
3. **اپائنٹمنٹ بناتا ہے** مریض سے منسلک
4. **انٹریکشن لاگ کرتا ہے** interactions ٹیبل میں
5. **کنفرمیشن کوڈ بناتا ہے** (فارمیٹ: APT-XXXXXX)
6. **مریض کو تصدیق دیتا ہے** اور کنفرمیشن کوڈ فراہم کرتا ہے

## دستیاب خدمات

### خدمات:
1. **مشاورت (Consultation)** - $19
   - عام طبی مشاورت
   - کوئی انشورنس قبول نہیں
   
2. **امیگریشن میڈیکل ایگزام** - $220
   - فزیکل ایگزام، خون کے ٹیسٹ، اور I-693 فارم شامل
   - ویکسین شامل نہیں (ہیلتھ سینٹر میں بھیجا جاتا ہے)
   
3. **پرائمری کیئر** - مختلف قیمتیں
4. **ماہر (Specialist)** - مختلف قیمتیں
5. **فوری نگہداشت (Urgent Care)** - مختلف قیمتیں
